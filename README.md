# react-ts-vite-template

- Language: TypeScript
- Framwork: React@18
- Router: React-router-dom@7
- Build Tool: Vite@6
- State Management: Zustand@5

## Environment

```bash
nvm ls
nvm use 20
```

## Install

```bash
npm install
```

## Development

Run with custom arguments.
Set your configuration in `/src/mock` folder, filename should be `[developer].ts`.

```bash
npm start [developer]
```
