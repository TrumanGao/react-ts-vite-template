# react-ts-vite-template

- Framwork: React
- Language: TypeScript
- Build Tool: Vite
- Router: React-router-dom
- State Management: Zustand

## Environment

```bash
nvm ls
nvm use 20
```

## Install

```bash
npm install
```

## Development

Run with custom arguments.
Set your configuration in `/src/mock` folder, filename should be `[developer name].ts`.

```bash
npm start [developer name]
```
